# GameCast

## Development

Clone this repository:

```sh
git clone https://github.com/jTronixDevelopment/gameCast.git
cd gameCast
```

Install dependencies:

```sh
npm install
```

Start the project at [`http://localhost:3000`](http://localhost:3000).

```sh
npm start
```

## Testing

```sh
npm test
```
